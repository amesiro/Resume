前端學習經歷

接觸程式語言

求學時有學習過自動化控制的課題，當時有些興趣，但並未往此方向發展。
就業後有協助導入ERP系統，對程式設計有興趣，但不知從何下手。在朋友的幫助下，開始學習網頁開發。

HTML, CSS

這是我在開始接觸前端的時候最先接觸到的編成，嚴格意義上來說這些不算是程式語言就是網頁的骨架以及樣式，但接觸後才發現原來平常我們瀏覽的網頁，是這樣去開發的。 從一個使用者到開發者，那種感覺真的很奇妙。

JS

讓網頁活起來的魔法，HTML的骨架、CSS的排版，最後加上JS的魔法，讓網頁鮮活起來。 之後也會慢慢的進入到框架學習的部分，目前的打算是選擇 Vue3 作為第一個接觸的框架，之後再持續精進 Nuxt3, 最後在專案進度允許的情況下使用 TS, 相信這對於以後轉換強型別的框架或者是 Debug 的部分都會受用無窮。

未來展望

網頁開發只是程式語言中的一小塊，在日新月異的不斷創新的科技業來說，現在的我只學到皮毛，要精進的項目還有很多，當然不要只有單機作戰的操演練習，還是要進入公司累積實務經驗，才會接觸到更多的領域。
